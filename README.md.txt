🏥 Northern Medical Complex - Enterprise Clinical DashboardThis project is an advanced Doctor/Hospital Management System designed for the Northern Medical Complex, Nowshera. The primary objective of this system is to digitize and automate daily hospital operations, patient appointment scheduling, and doctor availability management.  📸 System PreviewsLogin & OverviewDoctor DashboardAdmin Portal🚀 Key FeaturesDoctor Dashboard: Provides doctors with a centralized platform to manage their profiles, real-time availability, and daily appointment schedules.  Appointment Management: Seamless functionality to 'Approve' or 'Cancel' patient appointments instantly.  Patient Medical History: Allows easy tracking and retrieval of patient medical records and diagnostic history.  Doctor Lookup: Enables doctors to view the schedules of their colleagues for better inter-departmental coordination.  Modern UI: Built with Tailwind CSS to ensure a fully responsive, modern, and user-friendly interface.  🛠 Tech StackFrontend: React.js, Tailwind CSS, Axios  Backend: Node.js, Express.js  Database: MSSQL (Microsoft SQL Server)  ⚙️ How to Run LocallyClone the repository: git clone <your-repo-link>  Install Dependencies: Run npm install in both the frontend and backend directories.  Setup Database: Connect your MSSQL server and execute the provided SQL scripts to set up the database schema.  Run the Server:Start the backend: npm start  Start the frontend: npm run dev  📋 Additional Reference ViewsSchedule Lookup: assets/Screenshot_2026-06-25_153120.pngPatient Booking Portal: assets/Screenshot_2026-06-25_153215.pngDoctor Management (Admin): assets/Screenshot_2026-06-25_153323.png
